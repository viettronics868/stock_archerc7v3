WAN Connection Type Help

The Quick Setup is preparing to set up your internet connection, please choose one type below according to your ISP. The detailed description will be displayed after you choose the corresponding type.

Auto Detect - Let the router automatically detect the internet connection type provided by your ISP.

Dynamic IP - For Cable/DSL/Broadband connection which makes your computer immediately online without any setting or signing-in.

Static IP - Your ISP provide you specified IP parameters.

PPPoE/Russia PPPoE - You have DSL connection and are connecting via PPPoE (Broadband Connection) on your computer or existing router.

L2TP/Russia L2TP - In this type, you should fill in the username, password and IP address/Domain name of VPN Server provided by your ISP. Auto-Detect can't detect this connection type.

PPTP/Russia PPTP - In this type, you should fill in the username, password and IP address/Domain name of VPN Server provided by your ISP. Auto-Detect can't detect this connection type.
