PPPoEv6 Advanced Settings Help

MTU Size - The default MTU(Maximum Transmission Unit) size is 1480 bytes, which is usually fine. For some ISPs, you need modify the MTU. This should not be done unless you are sure it is necessary for your

Detect Online Interval - The default value is 0, you can input the value between 0 and 120. The Router will detect Access Concentrator online every interval seconds. If the value is 0, it means not detecting.

Use the following DNS Servers - If your ISP specify a DNS server IP address for you, click the checkbox, and fill the Primary DNS and Secondary DNS blanks below. The Secondary DNS is optional.Otherwise, the DNS servers will be assigned dynamically from ISP.

Primary DNS - Enter the DNS IP address in colon-hexadecimal notation provided by your ISP.

Secondary DNS - (Optional) Enter another DNS IP address in colon-hexadecimal notation provided by your ISP.

Note: The new advanced PPPoEv6 parameters will not take effect until you dial-up again.

Click the Save button to save your settings.

Click the Back button when finished.
