Add or Modify Share Folder Help

You can add or edit a share folder on this page

Volume Name - From the pull-down list, select which volume to save the file under.

Folder Path - Path to the folder is determined by where the file is saved.

Share Name - Name of the shared folder.

Allow Guest Network Access - Select the check box to allow the Guest Network to have access to the shared folder.

Enable Authentication - Select the check box to require users to log in with a username and password. (Note: You can create a new login in the Advanced USB Settings page. Otherwise, the username and password will be the same as your Web Management login info.)

Enable Write Access - Select the check box to allow users to make changes to the folder content.

Enable Media Sharing - Enable access to the folder.

Up - You can click the Up button to go into the upper folder.

Folder - Name of folders that is in current path.

Save - You can click the Save button to save your settings and page will be redirected to the ftp server configuration page.

Back - You can click the Back button to discard the settings and just go to the ftp server configuration page
