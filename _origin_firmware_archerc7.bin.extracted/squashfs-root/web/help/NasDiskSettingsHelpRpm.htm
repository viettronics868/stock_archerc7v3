Disk Settings Help

Configure the USB drive connected to the Router

Set up your Router as the file server:

  • Plug the external hard drive or USB flash drive into the router.
  • Click Scan to find the connected drive.
  • Click Safely Remove to eject the connected drive.
  • Check the Active box to enable file sharing.
  • Uncheck the Active box to disable file sharing.

In this section, you may view the Device Settings like ID, Volume, Capacity, Free Space and Sharing Status.

Volume - The volume name of the storage drive. Volume 1-8 is mapped to USB Port 1, Volume 9-16 is mapped to USB Port 2.

Capacity - The storage capacity of the storage drive.

Free Space - The available space of the storage drive.

Active - Indicates the sharing status of each storage volume.
