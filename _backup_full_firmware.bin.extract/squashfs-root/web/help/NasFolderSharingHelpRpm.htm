File Sharing Help

Sharing Account

  • You can use the login account or create a new one.
  • Use Login Account - The username and the password are the same as that of the login account. When choosing Use Login Account as shared account, the text boxes below cannot be edited.
  • Use New Account - You can create a new username and password.
  • Username - Type the username that you want to give access to the USB drive. The username must be composed of alphanumeric symbols not exceeding 15 characters in length.
  • Password - Enter the password in the Password field. The password must be composed of alphanumeric symbols not exceeding 15 characters in length. For security purposes, the password for each user account is not displayed.
  • Confirm Password - Re-enter the password here.

Sharing Settings

  • Network/Media Server Name: Name of the USB drive connected to the router.
    (Note: Network Connection is enabled by default.)
  • Enable: The Access Method is active if the Enable box is selected.
  • Access Method: Available methods to access the storage device.
  • FTP: FTP client(s) accessing from the home network.
  • FTP (Remote): FTP client(s) accessing from the Internet remotely.
  • Link: Access the storage device by clicking on the link or entering it into a web browser.
  • Sharing Mode - Select to share all or specific folders.
  • Share All - Share all active folders. Select the Require Login check box to require users to log in with a username and password.
    (Note: You can create a new login above. Otherwise, the username and password will be the same as your Web Management login info.)
  • Share Selected Folders - You may select which folders to share on the network.

Sharing Content

  • Create Share Folder - Click to create a new Share Folder.
  • Shared Name - Name of the shared folder.
  • Folder Path - Path to the shared folder.
  • Media Sharing - Enable/Disable access to a folder.
  • Volume Name - Volume name from the storage device.
